:::info 迁移提示
- 大部分 API 仅需 `cuda` → `hip` 前缀替换
- **warpSize 注意：** HIP 的 Wavefront 为 64 线程，CUDA 的 Warp 为 32 线程
- `cudaMallocHost` → `hipHostMalloc`（参数顺序不同！）
- `hipify-perl` 可自动完成 90%+ API 替换
:::
