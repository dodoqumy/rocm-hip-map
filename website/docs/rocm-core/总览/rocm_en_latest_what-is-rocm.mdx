Applies to Linux and Windows

ROCm is a software stack, composed primarily of open-source software, that provides the tools for programming AMD Graphics Processing Units (GPUs), from low-level kernels to high-level end-user applications.

[![AMD\'s ROCm software stack and enabling technologies.](_images/rocm-software-stack-7_2_1.png)&#123;.align-center style="width: 800px;"&#125;](_images/rocm-software-stack-7_2_1.png)

Specifically, ROCm provides the tools for [[HIP]](https://rocm.docs.amd.com/projects/HIP/en/latest/index.html "(in HIP Documentation v7.2.53211)"), OpenCL and OpenMP. These include compilers, libraries for high-level functions, debuggers, profilers and runtimes.

## ROCm components[\#](#rocm-components "Link to this heading")

ROCm consists of the following components. For information on the license associated with each component, see [[ROCm licensing]](about/license.html).

### Libraries[\#](#libraries "Link to this heading")

#### Machine Learning &amp; Computer Vision[\#](#machine-learning-computer-vision "Link to this heading")

  Component                                                                                                                                                                                   Description
  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- ------------------------------------------------------------------------------------------------------------------------------------
  [[Composable Kernel]](https://rocm.docs.amd.com/projects/composable_kernel/en/latest/index.html "(in Composable Kernel Documentation v1.2.0)")   Provides a programming model for writing performance critical kernels for machine learning workloads across multiple architectures
  [[MIGraphX]](https://rocm.docs.amd.com/projects/AMDMIGraphX/en/latest/index.html "(in MIGraphX v2.15.0)")                                        Graph inference engine that accelerates machine learning model inference
  [[MIOpen]](https://rocm.docs.amd.com/projects/MIOpen/en/latest/index.html "(in MIOpen Documentation v3.5.1)")                                    An open source deep-learning library
  [[MIVisionX]](https://rocm.docs.amd.com/projects/MIVisionX/en/latest/index.html "(in MIVisionX Documentation v3.5.0)")                           Set of comprehensive computer vision and machine learning libraries, utilities, and applications
  [[ROCm Performance Primitives (RPP)]](https://rocm.docs.amd.com/projects/rpp/en/latest/index.html "(in RPP documentation v2.2.1)")               Comprehensive high-performance computer vision library for AMD processors with HIP/OpenCL/CPU back-ends
  [[rocAL]](https://rocm.docs.amd.com/projects/rocAL/en/latest/index.html "(in rocAL Documentation v2.5.0)")                                       An augmentation library designed to decode and process images and videos
  [[rocDecode]](https://rocm.docs.amd.com/projects/rocDecode/en/latest/index.html "(in rocDecode documentation v1.7.0)")                           High-performance SDK for access to video decoding features on AMD GPUs
  [[rocJPEG]](https://rocm.docs.amd.com/projects/rocJPEG/en/latest/index.html "(in rocJPEG Documentation v1.4.0)")                                 Library for decoding JPG images on AMD GPUs
  [[rocPyDecode]](https://rocm.docs.amd.com/projects/rocPyDecode/en/latest/index.html "(in rocPyDecode v0.8.0)")                                   Provides access to rocDecode APIs in both Python and C/C++ languages

Note

[rocCV](https://rocm.docs.amd.com/projects/rocCV/en/latest/index.html) is an efficient GPU-accelerated library for image pre- and post-processing. rocCV is in an early access state. Using it on production workloads is not recommended.

#### Communication[\#](#communication "Link to this heading")

  Component                                                                                                                                             Description

  [[RCCL]](https://rocm.docs.amd.com/projects/rccl/en/latest/index.html "(in RCCL Documentation v2.27.7)")   Standalone library that provides multi-GPU and multi-node collective communication primitives
  [[rocSHMEM]](https://rocm.docs.amd.com/projects/rocSHMEM/en/latest/index.html "(in rocSHMEM v3.2.0)")      An intra-kernel networking library that provides GPU-centric networking through an OpenSHMEM-like interface

#### Math[\#](#math "Link to this heading")

  Component                                                                                                                                                                 Description
  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  [half](https://github.com/ROCm/half/)                                                                                                               C++ header-only library that provides an IEEE 754 conformant, 16-bit half-precision floating-point type, along with corresponding arithmetic operators, type conversions, and common mathematical functions
  [[hipBLAS]](https://rocm.docs.amd.com/projects/hipBLAS/en/latest/index.html "(in hipBLAS Documentation v3.2.0)")               BLAS-marshaling library that supports [[rocBLAS]](https://rocm.docs.amd.com/projects/rocBLAS/en/latest/index.html "(in rocBLAS Documentation v5.2.0)") and cuBLAS backends
  [[hipBLASLt]](https://rocm.docs.amd.com/projects/hipBLASLt/en/latest/index.html "(in hipBLASLt Documentation v1.2.2)")         Provides general matrix-matrix operations with a flexible API and extends functionalities beyond traditional BLAS library
  [[hipFFT]](https://rocm.docs.amd.com/projects/hipFFT/en/latest/index.html "(in hipFFT Documentation v1.0.22)")                 Fast Fourier transforms (FFT)-marshalling library that supports rocFFT or cuFFT backends
  [[hipfort]](https://rocm.docs.amd.com/projects/hipfort/en/latest/index.html "(in hipfort Documentation v0.7.1)")               Fortran interface library for accessing GPU Kernels
  [[hipRAND]](https://rocm.docs.amd.com/projects/hipRAND/en/latest/index.html "(in hipRAND Documentation v3.1.0)")               Ports CUDA applications that use the cuRAND library into the HIP layer
  [[hipSOLVER]](https://rocm.docs.amd.com/projects/hipSOLVER/en/latest/index.html "(in hipSOLVER Documentation v3.2.0)")         An LAPACK-marshalling library that supports [[rocSOLVER]](https://rocm.docs.amd.com/projects/rocSOLVER/en/latest/index.html "(in rocSOLVER Documentation v3.32.0)") and cuSOLVER backends
  [[hipSPARSE]](https://rocm.docs.amd.com/projects/hipSPARSE/en/latest/index.html "(in hipSPARSE Documentation v4.2.0)")         SPARSE-marshalling library that supports [[rocSPARSE]](https://rocm.docs.amd.com/projects/rocSPARSE/en/latest/index.html "(in rocSPARSE Documentation v4.2.0)") and cuSPARSE backends
  [[hipSPARSELt]](https://rocm.docs.amd.com/projects/hipSPARSELt/en/latest/index.html "(in hipSPARSELt Documentation v0.2.6)")   SPARSE-marshalling library with multiple supported backends
  [[rocALUTION]](https://rocm.docs.amd.com/projects/rocALUTION/en/latest/index.html "(in rocALUTION Documentation v4.1.0)")      Sparse linear algebra library for exploring fine-grained parallelism on ROCm runtime and toolchains
  [[rocBLAS]](https://rocm.docs.amd.com/projects/rocBLAS/en/latest/index.html "(in rocBLAS Documentation v5.2.0)")               BLAS implementation (in the HIP programming language) on the ROCm runtime and toolchains
  [[rocFFT]](https://rocm.docs.amd.com/projects/rocFFT/en/latest/index.html "(in rocFFT Documentation v1.0.36)")                 Software library for computing fast Fourier transforms (FFTs) written in HIP
  [[rocRAND]](https://rocm.docs.amd.com/projects/rocRAND/en/latest/index.html "(in rocRAND Documentation v4.2.0)")               Provides functions that generate pseudorandom and quasirandom numbers
  [[rocSOLVER]](https://rocm.docs.amd.com/projects/rocSOLVER/en/latest/index.html "(in rocSOLVER Documentation v3.32.0)")        An implementation of LAPACK routines on ROCm software, implemented in the HIP programming language and optimized for AMD's latest discrete GPUs
  [[rocSPARSE]](https://rocm.docs.amd.com/projects/rocSPARSE/en/latest/index.html "(in rocSPARSE Documentation v4.2.0)")         Exposes a common interface that provides BLAS for sparse computation implemented on ROCm runtime and toolchains (in the HIP programming language)
  [[rocWMMA]](https://rocm.docs.amd.com/projects/rocWMMA/en/latest/index.html "(in rocWMMA Documentation v2.2.0)")               C++ library for accelerating mixed-precision matrix multiply-accumulate (MMA) operations
  [[Tensile]](https://rocm.docs.amd.com/projects/Tensile/en/latest/src/index.html "(in Tensile Documentation v4.45.0)")          Creates benchmark-driven backend libraries for GEMMs, GEMM-like problems, and general N-dimensional tensor contractions

#### Performance[\#](#performance "Link to this heading")

  Component                                                                                                                                                                               Description
  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- -----------------------------------------------------------------------------------------------------------------------------------------
  [[ROCm Bandwidth Test]](https://rocm.docs.amd.com/projects/rocm_bandwidth_test/en/latest/index.html "(in rocm_bandwidth_test)")              Captures the performance characteristics of buffer copying and kernel read/write operations
  [[ROCm Compute Profiler]](https://rocm.docs.amd.com/projects/rocprofiler-compute/en/latest/index.html "(in ROCm Compute Profiler v3.4.0)")   Kernel-level profiling for machine learning and high performance computing (HPC) workloads
  [[ROCm Systems Profiler]](https://rocm.docs.amd.com/projects/rocprofiler-systems/en/latest/index.html "(in rocprofiler-systems v1.3.0)")     Comprehensive profiling and tracing of applications running on the CPU or the CPU and GPU

  [[ROCTracer]](https://rocm.docs.amd.com/projects/roctracer/en/latest/index.html "(in ROCTracer Documentation v4.1.0)")                       Intercepts runtime API calls and traces asynchronous activity

Note that [ROCprof Compute Viewer](https://rocm.docs.amd.com/projects/rocprof-compute-viewer/en/amd-mainline/) is in an early access state. Running production workloads is not recommended.
