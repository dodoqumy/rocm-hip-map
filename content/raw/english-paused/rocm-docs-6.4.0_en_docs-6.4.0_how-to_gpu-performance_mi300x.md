# AMD Instinct MI300X performance guides[#](#amd-instinct-mi300x-performance-guides)

2025-04-07

2 min read time

The following performance guides provide essential guidance on the necessary
steps to properly configure your system for AMD Instinct™ MI300X
accelerators. They include detailed
instructions on system settings and application [workload tuning](../rocm-for-ai/inference-optimization/workload.html) to help you
leverage the maximum capabilities of these accelerators and achieve superior
performance.

[AMD Instinct MI300X system optimization](https://instinct.docs.amd.com/projects/amdgpu-docs/en/latest/system-optimization/mi300x.html)covers essential system settings and system management practices to configure your AMD Instinct MI300X system for performance.[AMD Instinct MI300X workload optimization](../rocm-for-ai/inference-optimization/workload.html)covers steps to optimize the performance of AMD Instinct MI300X series accelerators for HPC and deep learning operations.[vLLM inference performance testing](../rocm-for-ai/inference/vllm-benchmark.html)introduces a preconfigured environment for LLM inference, designed to help you test performance with popular models on AMD Instinct MI300X series accelerators.
