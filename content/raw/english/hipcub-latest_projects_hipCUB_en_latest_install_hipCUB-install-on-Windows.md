<h1>Building and installing hipCUB on Windows<a class="headerlink" href="#building-and-installing-hipcub-on-windows" title="Link to this heading">#</a></h1>
<p>You can use <code class="docutils literal notranslate"><span class="pre">rmake.py</span></code> to build and install hipCUB on Microsoft Windows. You can also use <a class="reference external" href="./hipCUB-install-with-cmake.html">CMake</a> if you want more build and installation options.</p>
<p><a class="reference internal" href="hipCUB-install-overview.html"><span class="doc">Clone the hipCUB project</span></a>. <code class="docutils literal notranslate"><span class="pre">rmake.py</span></code> will be located in the <code class="docutils literal notranslate"><span class="pre">hipcub</span></code> root directory.</p>
<p>To build and install hipCUB, run:</p>
<div class="highlight-shell notranslate"><div class="highlight"><pre><span></span>python<span class="w"> </span>rmake.py<span class="w"> </span>-i
</pre></div>
</div>
<p>This command also downloads <a class="reference external" href="https://rocm.docs.amd.com/projects/rocPRIM/en/latest/index.html">rocPRIM</a> and installs it in <code class="docutils literal notranslate"><span class="pre">C:\hipSDK</span></code>.</p>
<p>The <code class="docutils literal notranslate"><span class="pre">-c</span></code> option builds all clients, including the unit tests:</p>
<div class="highlight-shell notranslate"><div class="highlight"><pre><span></span>python<span class="w"> </span>rmake.py<span class="w"> </span>-c
</pre></div>
</div>
<p>To see a complete list of <code class="docutils literal notranslate"><span class="pre">rmake.py</span></code> options, run:</p>
<div class="highlight-shell notranslate"><div class="highlight"><pre><span></span>python<span class="w"> </span>rmake.py<span class="w"> </span>--help
</pre></div>
</div>
