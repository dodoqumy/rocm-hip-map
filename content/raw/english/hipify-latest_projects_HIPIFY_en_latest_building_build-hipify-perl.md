# Building hipify-perl[#](#building-hipify-perl)

`hipify-perl`

is a perl-based script that heavily uses regular expressions, which is automatically generated from `hipify-clang`

. To generate `hipify-perl`

, run:

```
--perl
```

You can choose to specify the output directory for the generated `hipify-perl`

file using `--o-hipify-perl-dir`

option.
