# rocFFT API reference guide[#](#rocfft-api-reference-guide)

The rocFFT API reference information is divided into two sections. The [API usage](api.html#api-usage) topic describes the
datatypes and functions used to set up and implement the FFT in the hardware. The [API reference](allapi.html#api-reference) topic
provides a comprehensive listing of the classes and methods in the rocFFT library.
