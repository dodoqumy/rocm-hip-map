# rocPRIM prerequisites[#](#rocprim-prerequisites)

rocPRIM on Linux requires [ROCm](https://rocm.docs.amd.com/en/latest/). rocPRIM on Windows requires [HIP SDK for Windows](https://rocm.docs.amd.com/projects/install-on-windows/en/latest/).

rocPRIM uses [HIPCC](https://rocm.docs.amd.com/projects/HIPCC/en/latest/index.html) to build and run examples, tests, and benchmarks.

[CMake version 3.16 or later](https://cmake.org/) and C++17 are required on both Linux and Windows.
