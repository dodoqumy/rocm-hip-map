  <section id="hipfort-examples">
<h1>hipFORT examples<a class="headerlink" href="#hipfort-examples" title="Link to this heading">#</a></h1>
<p>Use the following examples to express Fortran 2003 (<cite>f2003</cite>) interfaces:</p>
<p><strong>Example 1</strong></p>
<div class="highlight-default notranslate"><div class="highlight"><pre><span></span>use iso_c_binding
use hipfort
integer     :: ierr        ! error code
real        :: a_h(5,6)    ! host array
type(c_ptr) :: a_d         ! device array pointer
!
ierr = hipMalloc(a_d,size(a_h)*4_c_size_t) ! real has 4 bytes
                                       ! append suffix &#39;_c_size_t&#39; to write &#39;4&#39;
                                       ! as &#39;integer(c_size_t)&#39;
ierr = hipMemcpy(a_d,c_loc(a_h),size(a_h)*4_c_size_t,hipMemcpyHostToDevice)
</pre></div>
</div>
<p><strong>Example 2</strong></p>
<div class="highlight-default notranslate"><div class="highlight"><pre><span></span>use hipfort
integer     :: ierr        ! error code
real        :: a_h(5,6)    ! host array
real,pointer :: a_d(:,:)   ! device array pointer
!
ierr = hipMalloc(a_d,shape(a_h))      ! or hipMalloc(a_d,[5,6]) or hipMalloc(a_d,5,6) or hipMalloc(a_d,mold=a_h)
ierr = hipMemcpy(a_d,a_h,size(a_h),hipMemcpyHostToDevice)
</pre></div>
</div>
<div class="admonition note">
<p class="admonition-title">Note</p>
<p>The Fortran 2008 (<cite>f2008</cite>) interfaces also overload <code class="docutils literal notranslate"><span class="pre">hipMalloc</span></code>, similar to the Fortran 2008 <code class="docutils literal notranslate"><span class="pre">ALLOCATE</span></code> intrinsic. For example:</p>
</div>
<div class="highlight-default notranslate"><div class="highlight"><pre><span></span>integer     :: ierr        ! error code
real        :: a_h(5,6)    ! host array
real,pointer :: a_d(:,:)   ! device array pointer
!
ierr = hipMalloc(a_d,source=a_h)       ! take shape (incl. bounds) of a_h and perform a blocking copy to device
</pre></div>
</div>
<p>In addition to <code class="docutils literal notranslate"><span class="pre">source</span></code>, there is also <code class="docutils literal notranslate"><span class="pre">dsource</span></code>, which is used if the source is a device array.</p>
</section>
