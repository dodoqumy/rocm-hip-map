<div class="contents">
<div class="textblock"><p>The ROCProfiler library provides profiling and tracing APIs for GPU compute applications.The APIs offer functionalities for profiling GPU applications in kernel, application, and user mode. The APIs also support no replay mode and provides the records pool support through a simple sequence of calls. This enables users to profile and trace in small steps. The sample codes demonstrate how to use the API calls for both profiling and tracing.</p>
<h1><a class="anchor" id="supported_amd_gpu_architectures"></a>
Supported AMD GPU architectures</h1>
<p>Here is the list of supported AMD GPU architectures:</p>
<ul>
<li>gfx900 (AMD Vega 10)</li>
<li>gfx906 (AMD Vega 7nm also referred to as AMD Vega 20)</li>
<li>gfx908 (AMD Instinct™ MI100 accelerator)</li>
<li>gfx90a (Aldebaran)</li>
<li>gfx94x (AMD Instinct™ MI300)</li>
<li>gfx1010 (Navi10)</li>
<li>gfx1011 (Navi12)</li>
<li>gfx1012 (Navi14)</li>
<li>gfx1030 (Sienna Cichlid)</li>
<li>gfx1031 (Navy Flounder)</li>
<li>gfx1032 (Dimgrey Cavefish)</li>
<li>gfx1100 (Navi31)</li>
</ul>
<h1><a class="anchor" id="known_limitations"></a>
Known limitations and restrictions</h1>

<ul>
<li>::ROCPROFILER_APPLICATION_REPLAY_MODE</li>
<li>::ROCPROFILER_USER_REPLAY_MODE</li>
</ul>
